# SGC Assignment

## Server (Node.js)
### To install module
`npm install`

### To build
`npm run build`

### To run development on local
`npm start`

### Available on
`http://localhost:8000/`


## Client (Vue.js) 
### First
`Change directory to 'web'`

### To install module
`npm install`

### To run development on local
`npm run serve`

### Available on
`http://localhost:8080/`
